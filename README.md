# 🧠 Memory Game

A simple and interactive Memory Card Game built using **HTML, CSS, and JavaScript**. The objective is to match all pairs of cards in the fewest moves and shortest time possible. This project demonstrates core front-end web development concepts and DOM manipulation.

## 🚀 Features

- 🎮 Interactive card matching gameplay
- 🎨 Clean and responsive user interface
- ⚡ Smooth card flip animations
- 🔄 Restart game functionality
- 📱 Responsive design for different screen sizes
- 🧩 Randomized card positions on each game

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/pralayakaverikalyani05-black/memory-game-mini-project.git
   ```

2. Open the project folder.

3. Double-click `index.html` or open it in your preferred web browser.

## 🎯 How to Play

- Click on any two cards to flip them.
- If the cards match, they remain open.
- If they do not match, they flip back after a short delay.
- Continue until all pairs are matched.
